# 旅游攻略生成器 - Claude Code Skill

## 这是什么

一个 Claude Code 自定义 Skill（slash command），输入出发地、目的地、天数等信息，自动生成包含**真实地图路线、行程安排、美食推荐、费用明细**的 HTML 攻略页面，支持 PC 和手机浏览。

## 功能清单

- 智能推荐 10-20 个候选城市并表格对比
- 基于高德地图的真实路线标注（中文地图）
- 多天路线按颜色切换显示
- 点击地图标记查看景点详情/美食/门票
- 每日行程卡片（时间/地点/干啥/吃啥）
- 电车充电规划（根据续航自动规划充电节点）
- 费用明细（高速/充电/门票/住宿/餐饮）
- 响应式布局（PC左右 / 手机上下）
- 支持部署到 GitHub Pages 生成公网链接

## 安装方法

### 方法一：直接复制文件
将 `trip.md` 复制到你的 Claude Code commands 目录：

**Windows:**
```
C:\Users\你的用户名\.claude\commands\trip.md
```

**Mac/Linux:**
```
~/.claude/commands/trip.md
```

> 如果 `commands` 目录不存在，手动创建即可。

### 方法二：项目级安装
如果只想在某个项目里用，放到项目根目录下：
```
你的项目/.claude/commands/trip.md
```

## 使用方法

在 Claude Code 对话中输入 `/trip` 加上你的需求：

```
/trip 从杭州出发 清明3天 电车455km 想去古镇
/trip 从上海出发 周末2天 油车 带娃 预算3000
/trip 从杭州出发 对比无锡和扬州
/trip 从南京出发 5天 想去山里徒步
```

## 自定义修改

打开 `trip.md` 可以修改：
- 默认输出目录（当前是 `D:\AI\`）
- 页面风格（渐变色、字体等）
- 地图瓦片源（当前是高德地图）
- 行程规划规则（如起床时间、续航余量比例等）

## 示例效果

生成的页面包含：
- 左侧：高德中文地图 + 路线 + 景点标记
- 右侧：天数切换按钮 + 行程卡片 + 费用明细
- 手机端自动切换为上下布局
